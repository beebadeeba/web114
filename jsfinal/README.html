JavaScript Mini-Game

This project is a simple browser-based game, created for my WEB 114 final. I chose to make this game because the assignment was to create an interactive webpage to demonstrate the skills we learned this semester. This game takes into account everything i've learned in ALL my classes this semester, including personal studying I did from independent Javascript sources.

Why I Chose This Game

I wanted to build something that I could show off to my friends and family, and I wanted something more engaging than just tic-tac-toe or a calculator.

This Project Includes

I think overall this project is a great endpoint for this semester, showcasing the experience I gained in fundamentals of front-end development. Even though the game is quite simple it combines everything I have practiced and showcases how Javascript can transform a website.
